1.0.0 (10.04.2024)
Behobene Fehler:
  • Es war möglich das Passwort zu sehen welches zur Verschlüsselung der Datenbank genutzt wurde.
Neuerungen:
  • Über die Checkbox ist es möglich anzugeben ob man sich in einem Workshop o.ä. befindet. Hierüber werden die Mehr / Minderstunden als 0 behandelt.
  • Es ist nun möglich eine Start und Endzeit zu erfassen und diese in eine SQLite Datenbank zu speichern.
  • Die Datenbank lässt sich nun verschlüsseln.
  • Wenn beim starten des Programm keine Datenbank gefunden werden kann, werden die Einstellungen geöffnet und man kann eine anlegen oder eine bereits angelegte DB hinterlegen.
  • Über Kommandozeilenargumente, ist es möglich die Start und Endzeit automatisch speichern zu lassen.
Änderungen:
  • Darstellungsanpassungen beim Erstellen einer Start und Endzeit über die Argumente.
  • Wenn man 2 Minuten vor oder nach Ende der Zeit aufhört, wird das noch als 0 Überminuten gewertet.
